# DEMAIN

A digital time capsule platform that allows users to preserve memories and deliver them to the future.

## Features

- Pixel-art themed postcard system
- Multiple custom themes
- Interactive writing experience
- Capsule scheduling
- Capsule management interface

## Technologies Used

- HTML
- CSS
- JavaScript

## Future Features

- Flask Backend
- Database Integration
- User Authentication
- Email Scheduling

## Status

Currently under active development.
